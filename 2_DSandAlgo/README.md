### Data Structures 
**Stack**
```JavaScript
    class Stack{
        constructor(){
            this.items = [];
        }
        push(item){
            this.items.push(item)
        }
        pop(){
            this.items.pop()
        }
        isEmpty(){
            return this.items.length === 0;
        },
        peek(){
            return this.items.length;
        }
    }
```
